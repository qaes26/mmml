Chapter 4: Computer Software

1. What are the step-by-step procedures that tell computers what to do called?
  • A. Hardware
  • B. Software
  • C. Peripherals
  • D. Circuits
Answer: B. Software Source: "These instructions, called programs or software..."
2. Why has the cost of software increased relative to hardware since the 1950s?
  • A. Hardware has become more expensive
  • B. Software is now mass-produced on assembly lines
  • C. Software development is slow, complex, and manual
  • D. There are fewer programmers today
Answer: C. Software development is slow, complex, and manual Source: "The software development process is usually slow, complex, error prone and expensive."
3. What is the "Software Crisis"?
  • A. The lack of hardware to run new apps
  • B. Organizations unable to develop software fast enough to keep up with changing conditions
  • C. The high cost of electricity for data centers
  • D. The disappearance of assembly language
Answer: B. Organizations unable to develop software fast enough to keep up with changing conditions Source: "Organizations are unable to develop new software applications fast enough to keep up with rapidly changing business conditions..."
4. Which level of software is considered the most basic abstraction closest to the hardware?
  • A. High-level language
  • B. Operating System
  • C. Assembly Code
  • D. Application Software
Answer: C. Assembly Code Source: "Assembly Code is the most basic of abstraction."
5. What characterizes RISC (Reduced Instruction Set Computer) architecture?
  • A. High power consumption and compatibility
  • B. Large number of instructions
  • C. Fast, low power, and fewer instructions
  • D. Used primarily in servers
Answer: C. Fast, low power, and fewer instructions Source: "RISC... (fast + low power + less number of instructions)"
6. What is the main function of the BIOS?
  • A. To edit Word documents
  • B. To provide a link between software and hardware
  • C. To compress files
  • D. To browse the internet
Answer: B. To provide a link between software and hardware Source: "The purpose of this system is to give software a link to the hardware."
7. Where is the BIOS typically stored?
  • A. On the Hard Drive
  • B. On a RAM stick
  • C. On a ROM-chip (Firmware)
  • D. On the CD-ROM
Answer: C. On a ROM-chip (Firmware) Source: "BIOS... is a small program permanently stored on a ROM-chip... called firmware"
8. What do drivers do?
  • A. Drive the cooling fans
  • B. Translate generic program instructions into instructions for specific hardware
  • C. Provide a graphical user interface
  • D. Protect the computer from viruses
Answer: B. Translate generic program instructions into instructions for specific hardware Source: "Drivers are small programs that translate the program’s generic instructions into instructions for a specific piece of hardware"
9. Which of the following is considered a Utility Program?
  • A. MS Word
  • B. Antivirus software
  • C. BIOS
  • D. Java
Answer: B. Antivirus software Source: "Antivirus utilities to protect your system from viruses."
10. Who developed the first Graphical User Interface (GUI)?
  • A. Microsoft
  • B. Apple
  • C. Xerox
  • D. IBM
Answer: C. Xerox Source: "Developed at a company called Xerox, in the 70s..."
11. Which interface uses a stylus to write on the screen?
  • A. GUI
  • B. Command Line
  • C. Pen-based interface
  • D. Conversational interface
Answer: C. Pen-based interface Source: "Pen-based interface... You use a stylus to write on the screen"
12. Which of the following is a High-Level Language?
  • A. Assembly
  • B. Java
  • C. Machine Code (0s & 1s)
  • D. BIOS
Answer: B. Java Source: "Examples: Java, Fortran, C++..."
13. What is a "Horizontal Market Application"?
  • A. Software for a specific industry only
  • B. General tasks used by almost everyone (e.g., Word Processing)
  • C. Software that runs on tablets only
  • D. Custom-made software for one company
Answer: B. General tasks used by almost everyone (e.g., Word Processing) Source: "Horizontal Market Applications: These are used to perform basic and general tasks, they can be used almost by everyone..."
14. An airline ticket reservation system is an example of what type of application?
  • A. Horizontal Market Application
  • B. Vertical Market Application
  • C. Utility Program
  • D. System Software
Answer: B. Vertical Market Application Source: "Vertical Market Applications... Examples: Ticket Reservation & Selling - Airline Industry"
15. What are Custom Applications?
  • A. Cheap, off-the-shelf software
  • B. Software written specifically for a company's unique needs
  • C. Free software available on the internet
  • D. Software that controls the mouse
Answer: B. Software written specifically for a company's unique needs Source: "Such companies decide to write (code) its own applications..."
16. What defines "Shareware"?
  • A. Completely free with no copyright
  • B. Source code is available to edit
  • C. Distributed for free/low fee for users to try before buying
  • D. Software that connects computers horizontally
Answer: C. Distributed for free/low fee for users to try before buying Source: "Shareware are programs... distributed to be used by users for free or by little registration-fee."
17. How does "Freeware" differ from "Public Domain"?
  • A. Freeware has no copyright; Public Domain does.
  • B. Freeware retains copyright; Public Domain has no restrictions.
  • C. Freeware costs money; Public Domain is free.
  • D. They are exactly the same.
Answer: B. Freeware retains copyright; Public Domain has no restrictions. Source: "Freeware: Software to which the author... still retains copyright (unlike public domain)..."
18. What is the key characteristic of Open Source software?
  • A. It is always free of cost
  • B. The source code is available for modification and redistribution
  • C. It does not have a license
  • D. It only runs on Linux
Answer: B. The source code is available for modification and redistribution Source: "Open source means that the source code is available to all potential users, and they are free to use, modify..."
19. Which software translates high-level human syntax into lower-level code?
  • A. The BIOS
  • B. The Driver
  • C. The Compiler
  • D. The Utility
Answer: C. The Compiler Source: "...the 'Compiler' translates almost human syntax into lower level code."
20. Which of the following is an Operating System?
  • A. MS Word
  • B. Linux
  • C. Java
  • D. BIOS
Answer: B. Linux Source: "Examples: ... Linux, HP Unix, Sun Solaris..."

  • A. MDR (Memory Data Register)
  • B. MAR (Memory Address Register)
  • C. ALU
  • D. Cache

31. What is Cache Memory?

  • B. Memory used for booting only
  • C. Faster, closer to CPU, stores frequently used info
  • D. Memory on a CD-ROM
Answer: C. Faster, closer to CPU, stores frequently used info Source: "Cache Memory... Closer to CPU & faster than RAM... Used to store information... used most often"
32. Which type of ROM can be erased and reprogrammed?

  • B. EPROM
  • C. RAM
  • D. Cache

  • B. By changing the polarity of magnetic particles

  • D. By using electron traps in crystals
Answer: B. By changing the polarity of magnetic particles Source: "...change the polarity of tiny magnetic particles on the surface... to record 0s & 1s."

  • A. North and South poles
  • B. Pits (dark) and Lands (shiny)

  • D. Black and White bars
